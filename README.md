# QA Testing Demo Project

This project demonstrates my approach to manual testing of a web application.

## 🔍 Objective
The goal of this project is to simulate real QA work:
- identify bugs
- create test cases
- think about edge cases
- ensure product quality before release

## 🌐 Tested Application
https://www.saucedemo.com/

## 🧪 Scope of Testing
- Login functionality
- Form validation
- Navigation
- UI behavior

## ✅ Testing Types
- Functional testing
- Negative testing
- UI testing

## 🛠 Tools
- Manual testing
- Test case documentation
- Bug reporting

## 📂 Project Structure
- test_cases.md → test scenarios and steps
- bug_report.md → identified issues

## 💡 QA Thinking
During testing I focused on:
- what can break
- invalid inputs
- user mistakes
- unexpected behavior
